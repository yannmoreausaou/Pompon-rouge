# Instructions pour le thème shopify :

**INSTRUCTIONS IMPORTANTES :** Tout les éléments créé sur les pages doivent être des éléments modifiables manuellement par l’utilisateur dans l’interface d’édition du thème sur shopify. Tout les éléments doivent être des modèles de bloc.

## Design :
Le design de la boutique doit être épuré/structuré. C’est une boutique qui vend un seul produit qui est une trousse de toilette de luxe qui à le design d’un pompon rouge de marin. Le design de la boutique doit donc rester simple tout en étant luxueux. Le thème est la mer et les marins.

## Couleurs :
- Bleu marine (#0F2D3D) pour titres, menu, footer, boutons secondaires 
- Rouge (#C62828) pour boutons principaux , accents, liens importants 
- Blanc cassé (#F8F6F2) pour fond principal du site 
- Sable (#E8DDC8) pour fonds de sections , cartes , encadrés 
- Gris ardoise (5B6770) pour texte courant, descriptions

## Polices :
- Titre = CINZEL
- Sous-titre = CINZEL REGULAR
- Texte = MONTSERRAT
- Boutons = MONTSERRAT SEMIBOLD

## Boutons :
*Principal :*
- Fond : #C62828
- Texte : #FFFFFF
*Secondaire :*
